# CourseProject_PP2023

## Запуск проекта в Linux

1. Склонируйте проект с использованием следующей команды в терминале:

    ```bash
    git clone https://github.com/hossameid7/CourseProject_PP2023.git
    ```

2. Перейдите в каталог проекта:

    ```bash
    cd ./CourseProject_PP2023/CourseProject
    ```

3. Запустите проект с помощью следующей команды:

    ```bash
    make run
    ```

4. Для отображения графиков используйте команду:

    ```bash
    make graph
    ```
